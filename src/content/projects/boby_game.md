# BobyGame：偵測身體動作來玩 Flappy Bird 遊戲
FIXME: (放遊戲畫面(影片))
## Concept Development

本專案結合 Mediapipe 與 OpenCV，基於 Python 開發一個利用肢體動作進行遊戲操控的系統。通過姿勢檢測技術，即時捕捉玩家的動作並將其轉換為遊戲中的操作指令，例如跳躍。此系統旨在提供更有趣的互動方式，實現肢體動作與數位遊戲的結合。
